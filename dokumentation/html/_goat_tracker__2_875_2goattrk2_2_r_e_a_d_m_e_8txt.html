outside the selector to cancel Song data Orderlist data A song can consist of up to subtunes For each subtune s each there is an orderlist which determines in what order patterns are to be played In addition to pattern there can be TRANSPOSE &REPEAT commands and finally there is a RST (RESTART) endmark followed by restart position.
