README for Sidplay2 w ~ June version (2005) Sidplay 2 is the second in the Sidplay series originally developed by Michael Schwendt. This version is written by Simon White and is cycle accurate for improved sound reproduction. Sidplay 2 is capable of playing all C64 mono and stereo file formats. Sidplay2/w is a Windows GUI for Sidplay 2
